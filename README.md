# json-to-nbt
A converter to convert json to nbt!

# install
```bash
npm i json-to-nbt -g
```

# Usage
### cli
```bash
json-to-nbt <file>
```
```bash
json-to-nbt <path to file>
```
### javascript
```js
const { jsonToNBT } = require('json-to-nbt');
```
### typescript
```ts
import { jsonToNBT } from 'json-to-nbt';
```
